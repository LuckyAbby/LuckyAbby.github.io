使用charles进行抓包

进行移动端开发的时候尤其是 ios 开发的时候，使用 charles 进行快速的抓包，提高我们的开发效率 。

抓取 HTTP 请求

1.安装完成之后打开 charles, 选择在“Proxy”菜单下的“Proxy Settings”子菜单。打开“Proxy Settings”对话框，可以设置 charles 的端口号，一般设置为 8888 。如图：

2.打开自己电脑的”网络偏好设置”，可以查看自己电脑的 IP 地址：

3.打开自己手机的手动代理，将服务器设置成自己电脑的 IP 地址，将端口号设置成自己在第一步中设置的端口号：

抓取 HTTPS 请求

1.安装证书

在菜单中选择 ”Help“，选择“SSL Proxying”—>“Install Charles Root Certificate，之后会进入到钥匙串管理界面。

2.设置证书信任

双击对应的证书，将“使用此证书时”的选项设置成始终信任。

手机安装证书

安卓这个时候已经可以直接代理了，但是 ios 还需要进一步设置：

1.手机已经设置好了代理。
2.在手机的浏览器中访问：http://charlesproxy.com/getssl，安装证书即可。
